# 💬 TCP & UDP Chat Simulator

## Overview

This project is a simple chat simulator developed in C to demonstrate the working of TCP and UDP communication protocols in Applied Computer Networks.

The application allows users to send messages using either TCP or UDP simulation and maintains a chat history of delivered messages.

## Features

* TCP communication simulation with acknowledgment (ACK)
* UDP communication simulation with packet loss behavior
* Chat history storage and display
* Menu-driven interface
* Sender and receiver communication simulation

## Technologies Used

* C Programming
* Computer Networks Concepts
* TCP Protocol Simulation
* UDP Protocol Simulation

## Learning Outcomes

* Understanding TCP reliability and acknowledgments
* Understanding UDP packet transmission and loss
* Message handling in network communication
* Menu-driven programming in C
